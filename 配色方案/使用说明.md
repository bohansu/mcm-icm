# MCM/ICM 官方配色与样式工具箱

这是一个为 MCM/ICM 数学建模竞赛准备的专业绘图配色工具箱。它整合了来自 `ggsci` (R) 和 `SciencePlots` (Python) 的官方标准配色与样式。

## 📂 文件结构说明

*   **`配色工具库.py`** (原 mcm_colors.py): 核心库文件。包含所有配色的 HEX 代码列表，以及加载样式的辅助函数。
*   **`样式配置文件/`** (原 mplstyles/): 包含 Matplotlib 的样式配置文件 (`.mplstyle`)。
    *   `期刊样式/`: 期刊样式 (Nature, IEEE 等)。
    *   `颜色样式/`: 配色样式 (Vibrant, High-Contrast 等)。
*   **`生成配色预览.py`** (原 preview.py): 运行此脚本生成配色预览图 `配色预览图.png`。
*   **`绘图示例.py`** (原 demo.py): 示例代码，演示如何使用本工具箱画图。
*   **`配色预览图.png`**: 配色概览图（已生成）。
*   **`参考素材/`**: 包含之前的地图、插图等素材。

## 🚀 快速开始

### 1. 引入库
确保 `配色工具库.py` 在你的项目目录下，或者在 Python 路径中。

```python
import matplotlib.pyplot as plt
import 配色工具库 as mcm_colors
```

### 2. 使用 Nature 样式 + NPG 配色 (推荐)

```python
# 一键加载 Nature 期刊样式和 NPG 配色
mcm_colors.use_nature_style()

plt.plot(x, y1, label='Group A') # 自动使用 NPG 红色
plt.plot(x, y2, label='Group B') # 自动使用 NPG 蓝色
plt.legend()
plt.show()
```

### 3. 手动选择其他样式和配色

```python
# 1. 加载 IEEE 样式
mcm_colors.use_style('ieee')

# 2. 获取特定的配色列表 (例如色盲友好的 Okabe-Ito)
colors = mcm_colors.get_palette('OKABE_ITO') # 或者直接用 mcm_colors.OKABE_ITO

# 3. 绘图
plt.plot(x, y, color=colors[0])
```

## ⚠️ 常见问题与注意事项

### 字体报错 (Font not found)
`use_nature_style()` 默认使用 `Arial` 或 `Helvetica` 字体（这是 Nature 的投稿要求）。
*   **现象**：运行报错 `findfont: Font family ['sans-serif'] not found` 或者生成的图中文字变成了方框。
*   **解决**：
    1.  **安装字体**：确保电脑安装了 Arial 字体（Windows 自带，Mac 自带）。
    2.  **临时修改**：如果实在搞不定字体，可以在代码中覆盖字体设置：
        ```python
        mcm_colors.use_nature_style()
        plt.rcParams['font.family'] = 'DejaVu Sans' # 使用 Matplotlib 默认字体
        ```
