# 2025 美赛工作区目录说明

这是一个标准化的比赛工作目录，旨在帮助团队协作时不混乱。

## 📁 目录结构

*   **`01_题目与要求/`**: 比赛开始后，第一时间把赛题 PDF 放这里。如果有翻译版本也放这里。
*   **`02_参考文献/`**: 存放下载的 PDF 论文。建议文件名格式：`[年份]作者-标题.pdf`。
*   **`03_数据/`**:
    *   `Raw_Data/`: **只读！** 存放原始下载的数据，永远不要修改这里的文件。
    *   `Processed_Data/`: 存放清洗、处理后的中间数据（CSV, Excel）。
    *   `Result_Data/`: 存放模型跑出来的结果数据。
*   **`04_代码/`**:
    *   `Q1_第一问/` ~ `Q4_第四问/`: 对应每个小问的代码。
    *   `Common/`: **公用工具箱**。我已经把**配色工具库**放进去了，直接 import 使用。
    *   `Temp/`: 草稿区。乱七八糟的测试代码放这里，确认好用的再移到 Qx 文件夹。
*   **`05_论文写作/`**:
    *   `Images/`: **只存放最终定稿的图片**。代码生成的图请直接保存到这里。
    *   `Tables/`: 存放 Excel 表格或 LaTeX 表格代码。
    *   `Sections/`: 如果用 Word 写作，可以按章节分文件写；如果用 LaTeX，这里放子 .tex 文件。
*   **`06_过程记录/`**: 每天的思路整理、白板照片、会议记录。

## 🚀 快速开始

### 绘图配色
在 `04_代码/Common` 中已经预置了配色工具。
在你的代码（比如 `Q1_第一问/solve.py`）中这样使用：

```python
import sys
import os
# 添加 Common 目录到路径
sys.path.append(os.path.abspath(os.path.join(os.path.dirname(__file__), '../Common')))

import matplotlib.pyplot as plt
import 配色工具库 as mcm_colors

# 使用 Nature 风格
mcm_colors.use_nature_style()
plt.plot([1, 2, 3], [4, 5, 6])
plt.savefig('../../05_论文写作/Images/result_q1.png') # 直接保存到论文图片文件夹
```
